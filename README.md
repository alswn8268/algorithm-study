# Algorithm Study

알고리즘 문제 풀이 저장소입니다.

## 디렉토리 구조

```
algorithm/
├── BOJ/           # 백준 온라인 저지
├── Programmers/   # 프로그래머스
├── LeetCode/      # LeetCode
└── SWEA/          # SW Expert Academy
```

## 파일 네이밍 규칙

- 백준: `{문제번호}_{문제이름}.ipynb` (예: `1234_소수찾기.ipynb`)
- 프로그래머스: `{문제이름}.ipynb` (예: `두_정수사이의_합.ipynb`)
- LeetCode: `{문제번호}_{문제이름}.ipynb` (예: `1_two-sum.ipynb`)

## 풀이 양식

각 노트북 파일은 아래 구조를 따릅니다.

1. **문제 설명** — 문제 링크 및 요약
2. **접근 방식** — 풀이 아이디어 및 알고리즘 선택 이유
3. **코드** — 최종 풀이 코드
4. **복잡도 분석** — 시간/공간 복잡도
5. **회고** — 어려웠던 점, 개선 사항
