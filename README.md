## 📋 커밋 컨벤션
[week{N}][day] 문제명 - 이름


| day 약어 | 의미 |
|----------|------|
| `mon` | 월요일 개념 |
| `tue` | 화요일 풀이 |
| `wed` | 수요일 풀이 |
| `thu` | 목요일 풀이 |
| `fri` | 금요일 모의테스트 |

**예시**
- `[week01][tue] 두 수의 합 - 김민주`
- `[week01][fri] 모의테스트 - 이은미`

**잔디 TIP 🌿**: 매일 풀이 후 바로 커밋하면 매주 최소 3개 이상의 커밋이 쌓여요. 꾸준히 하면 1년 잔디도 가능! 💪
